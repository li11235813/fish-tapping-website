# 小马哥摸鱼木鱼站

这是一个纯静态网站，直接打开 `index.html` 即可运行。

## 文件

- `index.html`：页面结构
- `styles.css`：样式与动画
- `script.js`：交互逻辑
- `assets/home-bg.jpg`：欢迎页背景图

## 功能

- 欢迎页：显示背景图与“欢迎小马哥来玩”
- 点击开始后进入木鱼页
- 左上角显示北京时间
- 右上角显示距离 18:00 下班倒计时
- 点击木鱼有动画，并提示“功德 +1”等文案
- 页面飘散小花朵
- 右下角显示“小马哥专属”

## 发布

把整个 `xiaomage-muyu` 文件夹上传到 GitHub Pages 或静态托管即可。
上传前把你的背景图文件命名为 `assets/home-bg.jpg` 放进站点目录。
