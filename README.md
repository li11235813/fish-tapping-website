# 摸鱼小网站 🎣

一个轻松愉快的摸鱼网站，帮助你在工作间隙放松身心！

## 功能特性 ✨

### 首页
- 绿色清新自然背景
- "欢迎小马哥来玩" 欢迎文字
- 点击"开始"按钮进入游戏

### 游戏页面

#### 🪵 木鱼敲击
- 可点击的立体木鱼
- 每次点击木鱼会有动画效果
- 点击后功德数量 +1
- 点击时有音效反馈（简单的木鱼声）
- 浮动的"+1"文字提示

#### ⏰ 时间与倒计时
- 左上角显示北京当前时间（实时更新）
- 显示距离下午18:00（下班时间）还有多少时间
- 实时倒计时功能

#### 🌸 装饰元素
- 页面散布8朵小花朵
- 花朵随风飘动，营造放松氛围
- 绿色渐变背景

#### 📊 功德计数
- 中央显示当前功德总数
- 点击木鱼后即时更新

#### 🏷️ 专属标签
- 右下角标注"小马哥专属"
- 为你的摸鱼时光增添个性

## 技术栈

- **HTML5**: 页面结构
- **CSS3**: 动画和样式
- **JavaScript**: 交互逻辑
- **Web Audio API**: 声音效果

## 使用方法

1. 打开 `index.html` 文件
2. 点击"点击开始"按钮进入游戏
3. 点击木鱼开始敲击，获得功德
4. 查看左上角的时间和倒计时，享受摸鱼时光！

## 响应式设计

网站支持桌面端和移动设备，自动适配不同屏幕尺寸。

## 浏览器兼容性

- Chrome / Edge (推荐)
- Firefox
- Safari
- 其他现代浏览器

## 提示 💡

- 声音效果需要浏览器支持 Web Audio API
- 时间显示使用北京时间（亚洲/上海时区）
- 倒计时会在18:00后自动切换到次日
